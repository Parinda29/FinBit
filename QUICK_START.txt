## 📞 Common Issues & Solutions

### Issue: Icons not showing
**Solution**: Make sure `@expo/vector-icons` is installed
```bash
npx expo install @expo/vector-icons
```

### Issue: Inputs not responding
**Solution**: Check that editable={true} (or undefined)
```typescript
<AuthInput editable={!loading} />
```

### Issue: Colors look different
**Solution**: Test on actual Android device, not just emulator
- Emulator colors may differ
- Try different theme settings

### Issue: Type errors
**Solution**: Make sure tsconfig.json is set up correctly
- Should extend Expo's config
- Check strict mode settings
